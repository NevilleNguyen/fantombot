# Fantom bot

Used to track events on Fantom chain.

# How it works
You have to update these following configures in `env/mainnet.json` config file:
- `min_staking_amount`
- `min_claim_amount`
- `min_transfer_amount`
- `telegram` fields `tokens` and `chat_id`

# Builds
Run command:
```go build ./...```

# Run 
Run command:
```go run main.go start```
